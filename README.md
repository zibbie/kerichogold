# Nevro-Shop v2 — Premium E-commerce Platform

Nevro-Shop v2 to nowoczesna, wysokowydajna platforma e-commerce zbudowana na fundamencie **Laravel 11**, **Livewire 3** oraz **TailwindCSS**. System został zaprojektowany z myślą o maksymalnej konwersji, optymalizacji SEO oraz gotowości na erę AI.

## 🚀 Kluczowe Funkcje Premium

### 1. Inteligentne Wyszukiwanie (Meilisearch)
Zintegrowany silnik **Meilisearch** zapewnia błyskawiczne wyniki typu *Instant Search* (wyszukiwanie w trakcie pisania). Obsługuje błędy ortograficzne, synonimy oraz zaawansowane filtrowanie, co drastycznie poprawia UX i ułatwia odnalezienie produktów.

### 2. Analityka Serwerowa i Śledzenie Marży
*   **GTM Server-Side:** Bezpieczne przesyłanie danych analitycznych bezpośrednio z serwera, omijające AdBlockery.
*   **Profit Tracking:** Automatyczna kalkulacja zysku (marży) dla każdego zamówienia widoczna w panelu administracyjnym Filament.

### 3. Płatności Odroczone (BNPL)
Pełna integracja z **PayPo** (przez Przelewy24). System oferuje dedykowane widgety na kartach produktów ("Kup teraz, zapłać za 30 dni") oraz deep-linking w koszyku, co zwiększa Average Order Value (AOV).

### 4. AI-Native & WebMCP
Pierwszy sklep w Polsce w pełni zintegrowany z protokołem **WebMCP** oraz standardem **ai.txt**. Pozwala to agentom AI (Claude, GPT) na autonomiczne przeszukiwanie katalogu i asystowanie użytkownikom w zakupach.

### 5. PWA (Progressive Web App)
Sklep można zainstalować jako aplikację na telefonie. Oferuje tryb **offline**, szybkie ładowanie dzięki Service Workerom oraz app-like navigation.

### 6. Mobile-First Polish
Interfejs zoptymalizowany pod kątem urządzeń mobilnych:
*   Dolny pasek nawigacyjny (Bottom Nav).
*   Sticky "Add to Cart" na karcie produktu.
*   Inteligentne klawiatury dla pól formularzy.

### 7. Wbudowane Narzędzia A/B Testing
System posiada gotowe scenariusze testów A/B zarządzane z poziomu admina (Filament):
*   Pasek postępu darmowej dostawy.
*   Optymalizacja tekstów przycisków CTA.
*   Testy kolorystyki elementów konwersyjnych.

## 🛠 Stos Technologiczny
- **Backend:** Laravel 11 (PHP 8.3+)
- **Frontend:** Livewire 3, Alpine.js, TailwindCSS
- **Database:** PostgreSQL / MySQL
- **Search:** Meilisearch (Docker)
- **Admin Panel:** Filament PHP v3
- **Integracje:** BaseLinker, Tpay, Przelewy24, InPost (Geowidget v5)
